MAC OS X 10.8 下安装jekyll

作者 : 
发布于 :  2013-04-08

本想在本机Macbook pro上装个jekyll,可是本机xcode自动安装了ruby 1.8.7版，导致无法通过当前版本的gem安装jekyll。使用 gem update --system 命令对gem进行升级，可是貌似没有升级成功。所以，我先装了个brew，通过brew安装了最新的ruby，最后通过新版gem成功安装了jekyll.
